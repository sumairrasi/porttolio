Recent Work

Mental-health-Assesment using MLOPS

The "Mental Health Assessment" project leverages machine learning and MLOps to detect, analyze, and address mental health challenges in the workplace. Designed to provide early warnings for potential issues, this system analyzes contributing factors and suggests personalized interventions, promoting a healthier work environment. Additionally, the project focuses on career optimization by integrating modern data science practices with real-world mental health challenges in a professional setting. I developed this project end-to-end, incorporating MLOps, deploying on Amazon EC2, and using Amazon S3 for storage.

Diary farm production forcast(MLOPS)

This project uses the SARIMAX model to forecast dairy farm production by analyzing historical data and incorporating external factors, with features for data collection, model training, forecasting, and visualization; it was developed end-to-end with CI/CD integration through GitHub Actions.

Sentiment Analysis of Twitter data using MLOPS

This sentiment analysis project leverages a Twitter dataset from Kaggle and utilizes the distilbert-base-uncased model, fine-tuned to accurately classify sentiments (positive, negative, neutral, or irrelevant) in tweets, with key features including tweet text and sentiment labels; I implemented the project end-to-end, covering advanced text classification and deploying with Docker for efficient application management.
